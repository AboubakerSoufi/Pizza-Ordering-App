
# Pizza Order System

A simple desktop pizza ordering application built using C# and Windows Forms with .NET Framework.

## Overview

This project allows users to customize and place pizza orders through a graphical user interface.  
Users can choose pizza size, crust type, toppings, and dining option while the total price updates automatically.

## Features

- Select pizza size:
  - Small
  - Medium
  - Large

- Choose crust type:
  - Thin Crust
  - Thick Crust

- Add multiple toppings:
  - Extra Cheese
  - Mushrooms
  - Tomatoes
  - Onion
  - Olives
  - Green Peppers

- Choose order type:
  - Eat In
  - Take Out

- Automatic total price calculation
- Order confirmation message
- Reset order functionality
- Dynamic order summary updates

## Technologies Used

- C#
- Windows Forms (WinForms)
- .NET Framework

## How It Works

The application updates the order summary and total price dynamically whenever the user changes:
- Pizza size
- Crust type
- Toppings
- Dining option

After confirming the order:
- A success message is displayed
- Order controls become disabled
